# DLW — SafeWatch

SafeWatch is a scalable, edge-first public safety intelligence system designed to improve early detection, responder coordination, and public advisories in urban communities. The system integrates lightweight AI-based anomaly detection with human-in-the-loop verification to ensure responsible deployment in real-world constrained environments.

Unlike traditional surveillance systems that rely on continuous cloud streaming, SafeWatch processes safety-relevant signals locally, transmits only structured incident metadata, and enables responders to verify alerts before broadcasting advisories to the public.
